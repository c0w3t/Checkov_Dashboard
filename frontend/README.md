# Checkov Security Dashboard - Frontend

## Quick Start

- Node.js 18+
- Cài đặt: `npm install`
- Chạy dev: `npm run dev`
- Build production: `npm run build`

## Features
- Dashboard tổng quan, Projects, Scans, Policies, Reports
- Upload và Scan tách biệt
- UI: React 18, TypeScript, Material-UI v5, Vite, Recharts

## Performance Optimization
- useMemo, useCallback, lazy loading, virtual scroll
- Dialog mở nhanh, filter policies tối ưu

## UI/UX
- Sidebar cố định, logo, layout 3 cột
- Tối ưu cho desktop

## Troubleshooting
- Xóa node_modules, cài lại nếu lỗi build
- Kiểm tra VITE_API_URL trong .env

## Tài liệu chi tiết
- Xem thêm hướng dẫn backend tại ../backend/README.md
